# App Modernization Intro

Welcome to the App Modernization Introduction workshop! In the workshop you will learn foundational container technologies for building modern, scalable, portable, and robust applications. You will learn about Containers, Kubernetes and OpenShift, and Helm.

## Pre-requirements

* IBM Cloud account,
* Docker Hub account,
* Access to Play with Docker using Docker account,
